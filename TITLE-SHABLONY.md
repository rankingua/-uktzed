# Банк шаблонів TITLE для сайту «Тарифник»

> Цей файл — джерело правди для генерації SEO-title. Коли додаються нові товари
> чи категорії, title пишуться ЗА ЦИМИ шаблонами (чергуючи), щоб уникнути
> машинного патерну. Користувачу НЕ треба щоразу пояснювати правила — вони тут.

## Загальні правила
1. **Довге тире «—»** по всьому сайту (не дефіс «-»).
2. **Довжина ≤ 56 символів** (комфортно для Google, рахувати через `[...str].length`,
   бо локаль контейнера рахує байти невірно). Жорстка межа — 60.
3. **Повний код** товару (не обрізати). Якщо код з варіаціями в дужках —
   для title береться частина до дужки: `kod.split("(")[0].trim()`.
4. **Чергувати шаблони** — сусідні товари (особливо однієї групи, напр. 8516)
   мусять мати РІЗНУ структуру. Не ставити однакові шаблони поспіль.
5. **Довгим назвам — короткі шаблони** (№3, 5, 6, 17, 22 нижче), щоб не виходити за межу.
6. **Унікальність** — жодних дублів title.

## Відмінки
- Шаблони з «для {Назва}», «розмитнення {Назва}», «імпорт {Назва}»,
  «класифікація {Назва}», «все про … для {Назва}» — потребують **РОДОВОГО** відмінка
  (для велосипедів, імпорт автозапчастин, розмитнення електроінструменту).
- Шаблон «Яке мито на {Назва}?» — потребує **ЗНАХІДНОГО** (= називний для неістот-множини):
  «на ноутбуки», «на термометри» (НЕ «на ноутбуків»).
- Решта — називний відмінок.

## 23 шаблони для ТОВАРІВ
(n = назва називний, k = код, g = назва родовий)

1. `{n} — код УКТЗЕД {k} | Довідник`
2. `{n} (УКТЗЕД {k}) — класифікація товару`
3. `{n} — код УКТЗЕД {k}`                         ← короткий
4. `{n} — класифікація за кодом УКТЗЕД {k}`
5. `Код УКТЗЕД {k} — {n}`                          ← короткий
6. `УКТЗЕД {k}: {n}`                               ← короткий
7. `{n} — опис та код УКТЗЕД {k}`
8. `{n} — класифікація товару | Код УКТЗЕД {k}`
9. `{n} — код УКТЗЕД {k}, мито та ПДВ`
10. `Код УКТЗЕД {k} для {g} | Мито та податки`
11. `{n} — розмитнення, код УКТЗЕД {k}`
12. `Який код УКТЗЕД для {g}?`
13. `Код для {g} за УКТЗЕД — {k}`
14. `Класифікація {g} за УКТЗЕД {k}`
15. `Імпорт {g}: код УКТЗЕД, мито та ПДВ`
16. `Розмитнення {g} — код УКТЗЕД {k}`
17. `{n}: мито, ПДВ та код УКТЗЕД`                 ← короткий
18. `{n} — митне оформлення та код УКТЗЕД`
19. `Все про код УКТЗЕД для {g}`
20. `{n}: код, мито та особливості класифікації`
21. `Яке мито на {n}? Код УКТЗЕД {k}`              ← знахідний (= називний мн.)
22. `{n}: визначення коду УКТЗЕД`                  ← короткий
23. `Код УКТЗЕД, мито та ПДВ для {g}`

## 5 шаблонів для КАТЕГОРІЙ
1. `{Назва категорії} — коди УКТЗЕД та класифікація`
2. `{Назва категорії}: перелік товарів з кодами УКТЗЕД`
3. `Коди УКТЗЕД для {Назва категорії} — опис і класифікація`
4. `{Назва категорії} — довідник кодів УКТЗЕД`
5. `Класифікація {Назва категорії} за кодами УКТЗЕД`

## Як застосовувати (для асистента в новій сесії)
- Прочитати цей файл ПЕРЕД написанням title для нових товарів.
- Для кожного нового товару: взяти коротку назву (n) + родовий (g) + чистий код (k),
  підібрати шаблон, що ЩЕ не йшов у сусідів, перевірити довжину `[...title].length ≤ 56`.
- Якщо задовгий — взяти короткий шаблон (3, 5, 6, 17, 22).
- Description теж унікальний, окремо (≈140–160 символів, ключі попереду).
